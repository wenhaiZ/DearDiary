#Dear Diary for Android#
*模仿iOS上的日记应用《素记》，做了一个安卓版的，算作学习安卓三个月的作业。
初步功能已实现，还存在一些问题。后续会不断优化代码，并结合学习的新内容为应用增添新功能。*
